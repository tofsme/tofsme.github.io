What is docker and why is so popular?

Sun, Jun 7, 2015 - 300 Words
Docker allows two processes run on the same computer with their own virtual operating system Docker provides tool chain and libs isolation for processes running in the same underlining OS. Notice that every process will also have its own RAM, disk and kernel. Consequence: Docker is cheaper than virtualization and applicable in most cases Virtualization replicates the guest OS and so waste resources. However, virtualization allows running different operating systems on the same hardware.
