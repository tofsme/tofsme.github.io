Why not declare global functions and variables in javascript?

You declare a function called MyFunction;

function MyFunction(){
    return "I'm happy!";
}

You have co-workers writing another piece of the software, or you need a third-part script;

function MyFunction() {
    return "I'm sad";
}

Somebody had the same name idea as you, now you have two functions with the same name and Javascript will not even complain about it, just keep the later;

What is gonna be, sad or happy? I bet sad…
